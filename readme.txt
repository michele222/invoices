# Invoices software

## How to install

1. Create database *assignment_db* in your local mysql instance
2. Import database data from file *database.sql*
3. Copy the folder *invoices* in your local php directory
4. Update the db connection parameters in page *include/parameters.php*, to reflect your configuaration

## How to use

1. Open a browser and go to [http://localhost/invoices/index.php](http://localhost/invoices/index.php)
